# encryption
Java 和 js 实现 AES 和 RSA 算法的互加解密

### 本地环境

```
JDK版本：1.8
Maven版本：3.5.0
```

### 说明

该项目是由 Maven 构建的标准的 Java 项目，内置了 Jetty 服务器组件。

**运行 cn.huangxulin.encryption.Application 中 main 方法即可启动内置的 Jetty 服务器**，默认端口是80

项目运行成功后，可通过浏览器访问 http://localhost:80/
